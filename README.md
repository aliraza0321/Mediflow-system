# Hospital Patient Management System

# Project Overview
This project is a Hospital Patient Management System developed using React, Node.js and SQL. The system manages patient records, hospital staff, appointments, medical information.

# Folder Hierarchy
frontend/ – React frontend application  
backend/ – Node.js backend API  
database/ – SQL schema and database scripts  
docs/ – Documentation and API references  

# Tech Stack
Frontend: React  
Backend: Node.js, Express  
Database:SQL  

# Installation and Setup Instructions
Clone the repository and install dependencies for frontend and backend and database.

# How to Run the Project
Frontend and backend servers must be started separately.

# Features
Patient registration  
Appointment booking  
Staff management  
Patient records management  
Staff availability checking  
Manage stock
Emergency treatment

# Limitations
Currently the system is designed for academic demonstration purposes.
